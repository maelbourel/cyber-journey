# __DNS / DHCP__

## __DNS (Domain Name System)__

Le DNS traduit un __nom de domaine__ --> __IP__  
Exemple :  
> google.com --> 142.250.74.14  
- DNS = annuaire d'Internet
- Sans DNS, on utiliserait que des adresse IP 
  

## __DHCP__

Le DHCP configure automatiquement une machine qui se connecte sur le réseau en lui donnant : 
- IP
- masque de sous réseau
- passerelle
- DNS
