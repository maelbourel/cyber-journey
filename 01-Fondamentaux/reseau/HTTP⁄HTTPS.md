# __HTTP / HTTPS__

## __HTTP__

- __HTTP__
- Protocole web
- Texte en clair
- Ports : __80__
- Vulnérable

## __HTTPS__

- __HTTPS__
- HTTP + chiffrement ( TLS )
- Ports : __443__
- Certificat
- Garantit ( confidentialité, intégrité, authenticité )

## __Différence importante__

__HTTP__ | __HTTPS__
-------|--------
non chiffré | chiffré
sniffable | sécurisé 
attaque faciles | plus résistant
